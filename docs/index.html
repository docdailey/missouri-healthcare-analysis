About This Analysis

This project provides a comprehensive assessment of Missouri's healthcare infrastructure, analyzing the distribution and accessibility of hospitals, Rural Health Clinics (RHCs), and Federally Qualified Health Centers (FQHCs) across the state.

Using geospatial analysis techniques and 30-minute drive time service areas (approximately 20-mile radius), we identified both redundancies and critical gaps in healthcare coverage. The analysis reveals significant opportunities for system optimization while highlighting essential facilities that must be preserved to maintain healthcare access in rural communities.

Data Sources

  • CMS Provider Enrollment Data (April & August 2024)
  • Missouri Hospital Association
  • HRSA Data Warehouse
  • Missouri Department of Health and Senior Services

Methodology

  • 100% geocoding success rate for all 478 facilities
  • Haversine distance calculations for service area analysis
  • 30-minute drive time standard for rural healthcare access
  • Consolidation opportunity identification using 10-mile clustering
